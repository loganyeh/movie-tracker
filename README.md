# Movie Tracker Web App

A responsive movie browsing web app built with React, TypeScript, Vite, Tailwind CSS, and the TMDB API.

## Features
- Browse trending, popular, and seasonal movies
- Search movies by title
- View detailed movie information

## Tech Stack
React, TypeScript, Vite, Tailwind CSS, TMDB API
